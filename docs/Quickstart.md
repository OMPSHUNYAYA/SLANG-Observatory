# ⭐ **SLANG-Observatory — Quickstart**

## **Structural Language (SLANG) — Tiny Deterministic Kernels**

Deterministic • Structure-Based • No Workflow • No Sequence • No Execution Dependency

Minimal kernels — full reference systems available for selected domains.

No workflows • No pipelines • No sequencing

---

## ⚡ **30-Second Proof**

Open any demo folder and run the script (no setup, no dependencies):

```
python demo/<demo_name>.py
```

Example:

```
python demo/SLANG-Invoice/slang_invoice.py
```

---

## 🔍 **What to Observe**

outcomes emerge directly from structure  
no workflow, sequencing, or orchestration required  
incomplete structure → no outcome (honest silence)  
complete structure → deterministic outcome  
identical structure → identical outcome  

---

## 🧠 **Conclusion**

Different inputs  
Different ordering  

→ Same outcome (if structure is identical)

---

## 🚫 **What SLANG Does NOT Do**

SLANG does not:

- execute workflows  
- depend on pipelines  
- require sequencing  
- enforce outcomes  
- guess missing structure  

---

## ✅ **What SLANG Does**

SLANG:

- resolves structure deterministically  
- reveals only valid outcomes  
- prevents unsafe or forced results  
- ensures reproducibility
